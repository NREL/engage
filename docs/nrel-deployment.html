NREL Deployment¶

NREL provides a hosted Enage application, please visit https://engage.nrel.gov/.

Any contributions to this project in repo will be reviewed, merged and finally deployed into NREL host.

AWS¶

This Engage web application is deployed on AWS.

Jenkins¶

This automation of continuous delivery (CD) of this project at NREL was setup via Jenkis.
