How to Use This Guide¶

Calliope 0.7¶

Engage utilizes Calliope for its backend server architecture. With the release of Calliope version 0.7 as a pre-release for user feedback, Engage has also integrated this new version into its staging environment. Feedback from users will be reviewed, and suggested improvements will be communicated to the Calliope team.

Please refer to Calliope 0.7.0 Documentation for more details: https://calliope.readthedocs.io/en/v0.7.0.dev4/
